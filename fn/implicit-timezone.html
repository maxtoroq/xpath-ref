fn:implicit-timezone

Returns the value of the implicit timezone property from the dynamic context.

Signature

fn:implicit-timezone() as xs:dayTimeDuration

Properties

This function is deterministic, context-dependent, and focus-independent. It depends on implicit timezone.

Rules

Returns the value of the implicit timezone property from the dynamic context. Components of the dynamic context are described in Dynamic Context Components.
